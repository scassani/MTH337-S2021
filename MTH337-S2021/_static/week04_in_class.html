Ex plot \(y=x^3+2x+3\) in \([0,1]\) using 100 steps . Use list comprehension to build x and y

[49]:

n=100 # number of steps

x=[k/n for k in range(n+1)]
y=[k**3+2*k+3 for k in x]

plt.figure(figsize=(10,10))
plt.plot(x,y,'r-.',lw=3)
plt.show()

isprime(n)

Ex produce the zig-zag plot, add your first last name in the title, and post it on discord in the zigzag-pattern channel

isprime(n): given a number n returns True if n is prime, False otherwise

  • given n loop through [2,n) with i (range, for loop) (to improve this you could look only in the range [2,sqrt(n)])

  • if n%i==0 -> number is not prime (return False)

  • otherwise the number is prime
