note on import¶

  • import math : import the whole module. to use objects from the module we need the math. prefix

  • from math import sin,atan,cos,pi : import only the objects mentioned, and we do not need a prefix

  • from math import *: import the whole module, do not need prefix (careful to overwrite risk)

  • import math as m : import the whole module, need the prefix m.

Monday 2/8/2021¶

From week 1: find all of the divisors of n

Steps: * grab all of the numbers that are smaller than n * pick a number d and check if it divides n * if it divides n, we put it somewhere * move to the next number and repeat steps 2,3

Tools: * need a way to get a range of numbers * conditional code ( check if something is true or not) * True False values ( boolean expression) * repetition (loops) * need a way to store all of these numbers (list)

Ex. write a function morethanfive(n) and returns a list with all the numbers \(\leq n\) that have five or more divisors

given n=100:

  • initialize a container ( empty list, mylist=[] )

  • for loop on all numbers \(\leq n\) (for i in range(1,n+1))

  • compute the divisors of i

  • if statement to check if they have 5 or more divisors

  • if yes, add to the list

[3]:

def morethan5(n):
    mylist=[]
    for i in range(1,n+1):
        numdiv=0
        for d in range(1,i+1):
            if i%d==0:
                numdiv+=1
        if numdiv>=5:
            mylist.append(i)
    return mylist

def divisors(n):
    myl=[]
    for d in range(1,n+1):
        if n%d==0:
            myl.append(d)
    return myl

def morethan5(n):
    mylist=[] # create empty list for storage
    for i in range(1,n+1): # loop on all numbers
        div=divisors(i) # find divisors
        numdiv=len(div)
        if numdiv>=5:
            mylist.append(i)
    return mylist

Logical operators and, or, not¶

  • exp1 and exp2 -> True if both are `True

  • exp1 or exp2 -> True if at least one `True
