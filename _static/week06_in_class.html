Summary

  • their size cannot be changed after creation
  • they can contain only objects of the same type
  • can be sliced, and indexed as lists
  • are mutable (like lists)
  • slicing creates a 'view' of the array (not a new array)
  • support vectorial operations (unlike lists)

subplot¶

plt.subplot(r,c,p):

  • r: number of rows
  • c: number of columns
  • p: position (starting at 1)

ex. produce a 3x2 subplot, $x\in [-\pi,\pi]$ use 101 points

  • $y=\cos(x)$ in position 1, color blue
  • $y=\sin(2x)$ in position 2, color green
  • $y=0.5 x^2 -x +3$ in position 4, color magenta
  • $y=\tan(x)$ in position 5, color yellow

post the image on discord, in the subplot channel

Use plt.title() for each subplot

Use plt.suptitle() for the major title which should be your first, last name

reminder plt.subplots_adjust(wspace = ,hspace =)

use np.linspace to generate $x$

plt.subplot(3,2,1)
plt.plot(x,y)
plt.subplot(3,2,2)
plt.plot(x,y)
