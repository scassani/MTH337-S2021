isprime(n): given a number n returns True if n is prime, False otherwise

  • given n loop through [2,n) with i (range, for loop) (to improve this you could look only in the range [2,sqrt(n)])
  • if n%i==0 -> number is not prime (return False)
  • otherwise the number is prime
